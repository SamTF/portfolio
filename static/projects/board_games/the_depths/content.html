Summary

The Depths: Karak Eight Peaks is an asymmetrical competitve dungeon crawler set in the Warhammer Fantasy universe, in the depths below the perenially fought-for Karak Eight Peaks stronghold in the mountains.


One player controls the stubborn Dwarves that must explore the dungeons to recover sacred artifacts from their ancestors. The other player controls the stalking and craven goblins that must stop the Dwarves.

Creation Process

Creation

This game was greatly inspired by one of my favourite boards: Space Hulk. Space Hulk is set in the Warhammer 40k universe and Games Workshop never made an equiavelent for the Warhammer Fantasy, so I decided to make my own version.

The rules are similar, with several changes made to allow for the new settings, for more unique character classes, and for items. The biggest change was the ruleset of the enemies (the goblins) which was fully changed to allow for the interesting and fun mechanics derived from the goblinsâ€™ cowardliness and in-fighting.

For this, I created these rules:

  • goblins need to always be at distance from the dwarves until they outnumber them 4:1, or are rallied by the standard bearer or warboss
  • goblins have a chance to hit each other when attacking, which will start in-fighting
  • in-fighting only ends when only one goblin remains or the warboss steps in

Game design and level design by me. Artwork and 3D Models by Games Workshop.
